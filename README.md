# Wordbee Name Generator

Streamlit app for generating:
- Wordbee Name
- AEM Name(s) for selected countries
- AEM URL(s)
- CSV export

## Run locally

```bash
pip install -r requirements.txt
streamlit run app.py
```

## Input flow

1. Paste copied AEM row(s)
2. Enter GTS ID
3. Select Marketing, Product, or both
4. Select countries when Marketing is selected
5. Click Generate
